## Крестики-нолики
Итоговое задание в рамках курса "Введение в ОС Unix".  
Автор - Ярмошенко Александр, ФИИТ УрФУ, МЕН-330802/ФТ-302.

### Описание
Игра крестики-нолики для одного игрока. Пользователь играет против бота. Игровое поле 3x3. Игрок выбирает, за кого играть (крестики или нолики). Крестики ходят первыми.  
Написан на bash с использованием `coreutils`.  
Работоспособность проверена на Ubuntu 24.04.3 + bash 5.2.21(1).

### Использование

1) Сохраните скрипт `tic_tac_toe_task.sh` в удобном месте.
2) Сделайте скрипт исполняемыми: `chmod +x tic_tac_toe_task.sh`
3) Запустите скрипт: `./tic_tac_toe_task.sh [-x|-o]`  
-x: играть за крестики. -o: играть за нолики.

### Примеры

```
./tic_tac_toe_task.sh -x
# Игра за крестики
```
```
./tic_tac_toe_task.sh -o
# Игра за нолики
```

### Алгоритм бота

1) Проверить, может ли бот выиграть в этот ход. Если да, сделать ход.
2) Проверить, может ли игрок выиграть в следующий ход. Если да, то заблокировать победу игрока.
3) Занять центр, если он свободен.
4) Пытаться занять углы (случайный из свободных).
5) Занять любую свободную клетку.

### Обработка ошибок

- Проверка аргументов: пользователь должен передать ровно один аргумент: `-x` или `-o`.
```
./tic_tac_toe_task.sh
Использование: ./tic_tac_toe_task.sh [-x|-o]
-x: играть за крестики. -o: играть за нолики.
```
```
./tic_tac_toe_task.sh -a
Использование: ./tic_tac_toe_task.sh [-x|-o]
-x: играть за крестики. -o: играть за нолики.
```
```
./tic_tac_toe_task.sh two args
Использование: ./tic_tac_toe_task.sh [-x|-o]
-x: играть за крестики. -o: играть за нолики.
```

- `set -euo pipefail` для строгого отслеживания ошибок.
